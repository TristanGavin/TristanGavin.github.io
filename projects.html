RVFireAid.com

Worked with a friend to build a website to provide survivors of the Southern Oregon Wildfire with an updated list of shelters, clothing, meals and services. We built a Node script using Google Sheets api to periodically query information from a community-maintained spreadsheet and used handlebars templating to populate our website with updated content.

Small Shell

Developed a command line shell for Unix operating systems in the C programming language. The shell creates processes, manages them, terminates them, and allows the user to query their status. It is essentially a mini operating system.

Recipe Generator
